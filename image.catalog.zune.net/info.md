# API v3.0
Example URLs:

http://image.catalog.zune.net/v3.0/image/{UUID}?resize=false&height={height}

http://image.catalog.zune.net/v3.0/image/{UUID}?resize=false&width={width}

http://image.catalog.zune.net/v3.0/image/{UUID}?resize=true&width={width}&height={height}
